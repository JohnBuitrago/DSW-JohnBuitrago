ETIQUETAS HTML

Tenemos incialmente que HTML es un lenguaje que permite insertar escritos y distintos tipos formas el cual puede ser comprendido por una persona asi como tambien es un lenguaje que reconoce la maquina y el navegador WEB; el cual a su vez interpreta este codigo y lo imprime en una pantalla. A estas etiquetas se les conoce como etiquetas de marcado.


Generalmente las todos los elementos de HTML estan divididos en tres partes lo cual al combinarse permite crear un elemento HTML; estas son las tres parte que lo componen:

  • Etiqueta de apertura: Es utilizada para indicar el lugar en el que empieza un elemento tiene como tal su corchete de apertura y cierre.

  • Contenido: Este es aquel que es visto por el usuariop final

  • Etiqueta de cierre: es similar a la etiqueta de apertura con la diferencia de que esta contiene una barra inclinada delante del elemento / .

ETIQUETAS BASICAS



<body> </body>


Es la parte principal la cual indica el contenido del cuerpo del documento HTML; esta etiqueta es fundamental en cualquier documento debido a que permite identificar en donde empieza y donde termina el documento.


<head> </head>


Se encuentra ubicado en la parte superiro del documento HTML y permite ubicar el titulo del documento asi como sus hojas de estilos, javaScript, CSS, Etc.


<div> </div>


Sirve para realizar agrupaciones de elementos y a su vez actua como plantilla para otros controles. Esta etiqueta nos permite elaborar un documento con estructuras definidas.


<a> </a>


Con esta etiqueta se pueden crear enlaces a otra paginas WEB; su atributo principal es href el cual nos permite colocar el enlace al que queremos que se direccione.


<strong> </strong>


Permite resaltar aquellas palabras que queremos que sobre salgan en los parrafos o dar enfasis en titulos para que sean mas visibles dentro de la pagina WEB.


<br> </br>


La etiqueta nos permite realizar saltos de linea, lo cual nos permitira realizar separaciones de parrafos o resaltar textos lo cual permite que sean mas leibles.


<h1></h1>.....<h6></h6>


Esta etiqueta nos permite dar formato al titulo; siendo <h1></h1> la etiqueta mas grande en HTML y <h6></h6> la mas pequeña.


<img> </img>


Este nos permite realizar la insecion de una imagen y funciona en complemento con src ya que este ultimo indica la ruta de en donde se encuentra el archivo.


<ol> </ol>....<li></li>......<ul></ul>


Estas etiquetas son utilizadas para la creacion de listas en donde <ol> </ol> y <li></li> sirven para crear etiquetas ordenadas; mientras que con la propiedad <ul></ul> se crean las listas de forma desordenadas.


<p> </p>


Con esta etiqueta podemos realizar agrupaciones de texto dentro de un parrafo determinado; es empleado con el fin de hacer que el contenido sea mas facil de leer y quede de una manera organizada.


<span> </span>


El objetivo de esta etiqueta es el de personalizar una parte determinada del texto; podemos darle atributos de color, estilo y tamaño.
