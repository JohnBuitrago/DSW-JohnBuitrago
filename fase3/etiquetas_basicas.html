ETIQUETAS HTML

Tenemos incialmente que HTML es un lenguaje que permite insertar escritos y distintos tipos formas el cual puede ser comprendido por una persona asi como tambien es un lenguaje que reconoce la maquina y el navegador WEB; el cual a su vez interpreta este codigo y lo imprime en una pantalla. A estas etiquetas se les conoce como etiquetas de marcado. 

Generalmente las todos los elementos de HTML estan divididos en tres partes lo cual al combinarse permite crear un elemento HTML; estas son las tres parte que lo componen:
  • Etiqueta de apertura: Es utilizada para indicar el lugar en el que empieza un elemento tiene como tal su corchete de apertura y cierre.
  • Contenido: Este es aquel que es visto por el usuariop final
  • Etiqueta de cierre: es similar a la etiqueta de apertura con la diferencia de que esta contiene una barra inclinada delante del elemento / .

Atributos y Valores
